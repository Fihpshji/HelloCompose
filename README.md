# HelloCompose

Навчальний приклад використання Jetpack Compose.

![Android CI](https://github.com/Fihpshji/HelloCompose/actions/workflows/android-ci.yml/badge.svg)

## Як запустити

1. Встановіть Android Studio та SDK (API 26+).
2. Клонуйте репозиторій:
```
git clone https://github.com/Fihpshji/HelloCompose.git
```
3. Відкрийте проект у Android Studio → Sync → Run на AVD або реальному пристрої.

## Структура проєкту

- `app/src/main` — основний код додатку
- `.github/workflows` — CI конфігурація
- `.github/ISSUE_TEMPLATE` — шаблони для задач

## Технології

- Kotlin
- Jetpack Compose
- Material3
- GitHub Actions CI
